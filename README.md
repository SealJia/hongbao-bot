# 外卖红包微信机器人
> 用微信机器人抢美团和饿了吗外卖最大红包

## 功能
- 像机器人微信转发饿了吗美团手气红包,提示后填写手机号码即可领取最大红包
- 你只会领取你发送的红包，能不能领到最大红包看命

![](http://ovs5x36k4.bkt.clouddn.com/QQ20180227-0.png?imageView2/2/w/400)

## 环境
node.js v8.9.4

## 运行
修改`app.js`文件中的微信号为你的微信号
```
npm install
node app
```
